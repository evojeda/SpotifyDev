🎧 SpotifyDev

Interface inspirada no Spotify, desenvolvida como projeto de prática utilizando HTML, CSS e JavaScript.

🔗 Acesse o projeto online:
https://evojeda.github.io/SpotifyDev/

📌 Sobre o projeto

O SpotifyDev é uma interface web inspirada no layout do Spotify, criada com o objetivo de praticar estruturação de páginas, estilização de componentes e organização visual de layouts modernos.

O projeto simula elementos comuns de um player de música, como:
menu lateral;
playlists;
navegação visual semelhante à plataforma original.

🧠 Tecnologias utilizadas
HTML5;
CSS3;
JavaScript;
Font Awesome (ícones);
GitHub Pages (deploy do projeto).
